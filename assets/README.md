# Assets

Place the following files here:

- `zense-logo.png` - Zense logo for signature (height: 36px)
- `icon-vimeo.png` - Vimeo icon (height: 18px)
- `icon-instagram.png` - Instagram icon (height: 18px)
- `icon-linkedin.png` - LinkedIn icon (height: 18px)

These are served via GitHub Pages at:
https://zense-gmbh.github.io/mail-signature/assets/

PNG format recommended for best email client compatibility.
SVG does not work reliably in all email clients.
